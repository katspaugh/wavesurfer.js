Initialisation script Example

Initialising from HTML element

Add the html-init.js file and then use this html tag. It will be initialised to a wavesurfer.js instance on page load.

<wavesurfer
    data-wave-color="violet"
    data-progress-color="purple"
    data-url="../media/demo.wav">
</wavesurfer>
