Declarative Renderer with Auto-updates

This demo shows how the new DeclarativeRenderer automatically updates the cursor and progress bar when state changes - no manual render() calls needed!

How It Works

// 1. Create reactive state
const { state, actions } = createWaveSurferState()

// 2. Create declarative renderer (sets up effects automatically)
const renderer = new DeclarativeRenderer(container, state, {
  container: '#waveform',
  height: 128,
  cursorColor: '#4CAF50',
  progressColor: 'rgba(76, 175, 80, 0.3)',
  autoScroll: true
})

// 3. Just update state - UI updates automatically!
actions.setCurrentTime(50)  // ‚Üê Cursor moves automatically!
actions.setPlaying(true)     // ‚Üê Auto-scroll activates!

// No manual render() calls needed!
// No timers to manage!
// Clean and declarative!
